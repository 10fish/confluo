Detailed Description

template<typename T, uint32_t sampling_rate = 128>
class confluo::delta_encoded_array< T, sampling_rate >

Delta-encoded array class. Stores sorted integer data encoded using delta-encoding.

Template Parameters
TThe data type (can only be integers)
sampling_rateThe rate at which data is sampled.
