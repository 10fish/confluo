Detailed Description

template<typename T>
class confluo::storage::read_only_encoded_ptr< T >

Should be created by a swappable_encoded_ptr<T> or by copy of another read_only_encoded_ptr<T>. It holds a pointer to a reference count of a swappable_encoded_ptr<T>.
