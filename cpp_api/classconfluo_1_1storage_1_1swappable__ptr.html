Detailed Description

template<typename T>
class confluo::storage::swappable_ptr< T >

The lifetime of a swappable pointer must exceed the lifetime of any copies created by it, since copies rely on reference counts allocated in the original pointer.

After the internal pointer is set for the first time it can never be null.
