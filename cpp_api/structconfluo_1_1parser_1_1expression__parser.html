Detailed Description

template<typename I>
struct confluo::parser::expression_parser< I >

Parser of expressions

Grammar: 

* <expression>::=<term>{"||"<term>}
* <term>::=<factor>{"&&"<factor>}
* <factor>::=<predicate> | "!"<factor> | '('<expression>')'
* <predicate>::= <identifier>"<"<value> | <identifier>">"<value> ...
* <identifier>::= "alnum*"
* <value>::= "alnum*"
*
